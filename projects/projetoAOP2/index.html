Webiblioteca

Este blog é sobre Livros. Ele contém as páginas HOME, RESENHAS, LEITURAS, LOJA e CONTATO. (Este blog precisa de recursos JS para ser totalmente funcional. As funcionalidades dependentes foram desenvolvidas apenas como modelo).

  • HOME: contém a descrição do blog e imagens que dão acesso às demais páginas.
  • RESENHAS: dá acesso às resenhas de livros publicadas pelo autor do blog. (Os links de cada livro nesta página são direcionados a uma mesma página-modelo de resenha).
  • LEITURAS: permite que o autor mantenha um histórico de leituras.
  • LOJA: encaminha o usuário para o link do livro desejado em uma loja externa. (Podendo ser implementada uma loja própria).
  • CONTATO: disponibiliza ao usuário formas de entrar em contato com o autor.
